# Ejercicio en el lenguaje de Java

### Enunciado
El consulado de un país debe procesar todas las **solicitudes** de visado que se reciben cada día de la semana. Cada solicitud
recoge el código (es un número consecutivo), nombre del solicitante, número de pasaporte, fecha tentativa de viajar,
cantidad de semanas a permanecer en el país y el motivo de su solicitudes. Las solicitudes de visado pueden ser de dos
tipos, visa de **turista** o visa de **trabajo**. Para la solicitud de visa de turista, se recogen además el lugar de **hospedaje** (hotel,
casa de renta, casa de un conocido) Para la solicitud de visa de trabajo se recoge el nombre de la entidad contratadora, el
tipo de trabajo a realizar, la tarifa horaria a devengar y la cantidad de horas a laborar por cada día de la semana.
Para el pago del visado por servicios consulares se cobra 60 dólares, o el equivalente en moneda nacional según el cambio
establecido por el banco. Adicionalmente, para las visa de turismo se incrementa un 0.05 por ciento de ese pago, por cada
mes de estadía. Para las visas de trabajo, el incremento sería de 0.05 del salario semanal a percibir durante la estadía.

### El programa debe permitir:
a) Calcular el importe total de los pagos a percibir por concepto de visa de turismo
b) Obtener la visa de trabajo que debe pagar mayor cantidad de dinero
